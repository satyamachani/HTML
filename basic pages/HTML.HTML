      NAME ("TEXT"): It specifies a mark with in the document to act as destination.

      Anchor should be written in this following type.

     < IMG SRC>: It specifies the location of the image file and we can include images in

					FONT
Types:  i. FONT COLOR = "RED"  - It changes the font color.  Ex: < FONT COLOR="RED">  < /FONT>
       ii. FONT SIZE = "4PT"   - It changes the font size.   Ex: < FONT SIZE="5PT">   < /FONT>
      iii. FONT FACE = "ARIAL" - It changes the font face.   Ex: < FONT FACE="ARIAL"> < /FONT>


      	

					FRAMES

	Using Frames we can controll the layout of the browser window by segregating the window 
  to rectangular sections & loading seperate HTML files into each section of the window.It 
  specifies the file that should be displayed in frame.
	< FRAMESET>   < /FRAMESET>
  Divides the pages in to frames COLs & ROWs, Means divides document into COLs & ROWs.
	< FRAMESET COLS="40 %, 60 %">
	< FRAMESET ROWS="50 %, 50 %">
  SRC: Specifies the file or url.  < FRAME SRC="color2.bmp">
	
  NORESIZE: Indicates the size of frame is fixed that is cannot alter by users.
  SCROLLING: Shows scroll box it can take values YES, NO or AUTO.
  
					FORMS
	
	This special feature users can enter information in a webpage and send it to the server.
  ACTION: It specifies the location where the form data is input.
  INPUT : TYPE = TEXT	Shows single line text
	  TYPE = RADIO  Indicates radio button
	  TYPE = CHECKBOX Creates a checkbox
	  TYPE = SUBMIT	passes data to server
	  TYPE = RESET	Clears the data
	  TYPE = HIDDEN That won't be displayed			TOP BOTTOM
	  TYPE = PASSWORD Displays the stars in the field

	< SELECT>  < /SELECT>  Give the options and close

  NAME = Uniquely identify the tag
  SIZE = Identifies the number of items visible
  Multiple = Select multiple items
 
	< OPTION>  Defines the item of pull down menu
	< TEXTAREA>  < /TEXTAREA> Specifies the common lines for text field.
  NAME = Name of the TextArea  
  ROWS = Number of rows
  COLS = Number of coloums to displayed.

					TABLES

	It can contain any kind of data i.e., it can include links, paragraphs, all headers 
  levels,another table, performatted text, images & so on.
  < TABLE> 	We can write in only these tags.  
  < CAPTION> SBI BRANCHES < /CAPTION>  Heading for the table.
  < TR> < /TR> Table Rows
  < TH> < /TH> Heading for the column
  < TD> < /TD> Individual table elements < /TABLE>
  < CELLPADDING> Space between border & contents of cell.
  < CELLSPACING> Spacing between cells
