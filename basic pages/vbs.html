Introduction To VB Script

Visual Basic Script is a member of microsoft Visual basic family of development product. It is used to control contents and objects in a HTML pages. This extends HTML into something more than a page-formatting language.

VBScript acts as both server-side programming language and a client side programming language.It is a language that can be interpreted and executed by a browser. Java and JavaScript are examples of this programming. It performs all the work on the website computer.
